# 写作纪略

总算在今年（2021）的陪产假写完第一稿，也算感恩老婆生孩子吧。当然以后有时间要继
续修改。

本故事从开始落笔到初稿完成，花了一年多时间，因为中间一度暂停创作。
在完成整体构思、背景设定与主要人物设定后，在 2020 年写完一半，除了引入危机故
事，揭秘许梦龙身世，另外主要是写白灵素女的心理转变历程。一个由 DNA 刻意创造的
人类女孩，像个撒娇的普通女孩，从开始与父亲程思归子有些矛盾，到接受自己的身份。
只有让她自己打通了这个心结，才能上天寻找虫洞进入高维空间，与白蛇元神融合。

所以故事前半部分写到白灵素女要去集训宇航员。到这里也由于时间原因，也或有思路卡
壳的原因，就暂停了。

直到 2021 年中才有时间继续按原计划写完后半部分。故事里时间线直接跳到 2 年后。
毕竟在这故事里没有多余篇幅来写白灵素女的训练，以及许梦龙在程教授门下研究生生涯
过程。故后半部分开始就是主角团乘宇宙飞船上天执行任务。

在后半部分写作过程中更常遇一些卡壳，因为涉及一些有关宇航、天体的科技知识，需要
查资料尽可能将细节描写合理点。浑沌号与（智能控制系统）扬洋的名字是临时想到，高
级机器人不一定要有实体，然后又有意类比神可能也像是机器人，譬如元神可以像软件一
样独立存在，只不过硬件是人类生物体，于是白灵素女上面祭献，白蛇就能施展法力工作
了。

当然也不能说白灵素女就死了，毕竟与白蛇元神融合后还可能保留白灵素女的记忆。为了
故事人物的延续性，祭献融合的白蛇，仍用白灵素女之名称之。

之前上天送行时突然想到只让母亲来送会好点，否则白尘如梦几乎整片都只与程思归子同
框出现，就有成附庸的嫌疑。而且想了个理由，那时间点有重要的国际会议要开，不能给
爱女送行，也是合理的。因为有父女错别这个遗憾，在白灵素女成神后，刚抱起地球时，
加了一段托梦的情节。前期许梦龙一直在强调做梦见到亲娘，却没有找到有合适的机会来
叙说他的梦境，也觉得不一定有必要。于是将梦会的情节转移给程思归子夫妇，表达另一
种感情也行。

初期构思时也没能想清楚如何寻找到虫洞，所以写到那里时有免又卡了会。总不能写闲逛
闲逛就恰好撞入了虫洞吧。这算是关键转折点，没法回避。所以就有殒石先锋这一小节的
情节。当大家的关注点不在寻找虫洞了才意外进入虫洞，当然也有小小的舍生取义的渲染
。

后面的金字塔三花聚顶是早有构思好的，算是核心高潮吧，神话与科幻的结合点也在这里
体现。未来社会，信息通讯最关键，通讯传达到位后，对神来说，拯救地球不是个事。这
不仅是人类集体的协作，还是人神之间的合作。我只惶恐，仅凭文字能否表达出那个紧迫
感。

完美计划紧接着隐藏计划的助推变轨，然后张宝山铁游萍浮隔着地球遥相守望算是另一个
情感高潮。当然最开始的空间站不会只有他们两人，所以为了充实空间站上的故事，不免
加了个龙套，印度唐三，略灰谐的角色，在紧张的气氛中调剂。而且让印度人多生的设定
应该合理，与易与张宝山产生共鸣。然后为了争取最后的站岗，与空间站控制系统莫思尔
有冲突，用投票解决，这是写作时突然想到的，希望能深化表现人物。铁游萍浮跟进站岗
，也稍改了设定，一开始张宝山并不知情，因为尘埃阻断通讯。三年后突然或终于联系上
，应该更能渲染那种悲喜感。

被抱走的地球在那个特殊空间发生的故事，写作中又为玛雅金字塔加戏了。后面它变身月
球上的核实验室遥控中心，牺牲掉月球上的实验室，换来短暂（十余天）的月亮照明，表
达灯塔的愿望，在黑暗中光明的意义。海上生明月，天涯共此时的真实写照。

然后写作时还突发灵感，发掘二青与扬洋这对机器人的 CP 感。

结尾其实很难收，原计划是在想日本海洋城市奠基仪式上许梦龙作为嘉宾与救世英雄发表
演讲，点题，感念母亲。但是写着写着觉得很尬，说教成分严重。不过仍是硬着头皮先写
完，然后全部删除，留白处理，直接结束。反正有 git 的版本控制记录，删掉的整场台
词需要时也能找回来。

从整体看，删掉末尾演讲，可能与之前加的一段情节也有着。许梦龙李含珠刚回地球时直
接跳伞落到布达拉宫广场，迫不及待要见（人类的）母亲。这已经让他们出现在公众面前
了，并被惊为天人。只是在宗教人士面前，许梦龙又无法多讲，只能语焉不详搪塞过去，
也避免了演讲的尴尬。

基于这个故事，留了两个尾巴，可为后续提供素材。一是开采稳定到小行星轨道的尼比鲁
星，在花絮中大家将它重命名为万岁星。打算让许梦与李含珠再出征。

另一个是铁游萍浮与张宝山的虐恋，可以结合海洋城市发挥。也因此删了最后的奠基仪式
的演讲戏份，只保留场景留白，有必要的话可在续作故事中稍作补充。
